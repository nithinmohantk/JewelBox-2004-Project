****************
Reg Control v1.0
****************

This control helps perform registry functions in vb. This control is free and is not to be sold. You may distribute it freely.

Place Registry Control.ocx and Reg Control.chm in the system directory. Place the control on a form in vb by selecting registry from the components list (right click on the tool bar in vb and select Components). For help for this control you can press F1 at any time.
